# Future Album（未来アルバム）

## 目的
「親に会いたい」「話を聞きたい」と思っていても、
「面倒」「恥ずかしい」「何をしたらいいか分からない」で行動できない問題を解決する。
未来アルバムは「親との時間を作るための口実」を提供するアプリ。

## コンセプト
「今度やろう」を「今日やろう」に変える。

## ターゲット
第一ターゲット：開発者自身（親孝行したいが行動できていない）
拡張ターゲット：同じ悩みを持つ人

## コア体験
1. ルーレットを回す
2. 今日のミッションが決まる
3. 実際に行動する
4. 写真を記録する（3段階）
5. 一言メモを書く
6. イラストに色が付く
7. アルバムとして残る

## 方針
- シンプル最優先
- 登録不要
- スマホファースト
- 個人情報は最小限
- MVPを優先
- 完成を遅らせる提案は禁止

## 技術方針
- HTML / CSS / JavaScript のみ
- サーバーなし
- ローカルストレージ保存
- 写真はブラウザ内保存
- API不使用
- GitHub Pages で公開予定

## MVP機能
1. ルーレット（ミッション生成）
2. 未来スロット（同時進行3枠）
3. 固定イラスト表示
4. 色付けシステム（行動で少しずつ色が付く）
5. 写真3段階アップロード
6. 一言メモ
7. アルバム保存・表示

## 禁止
- SNS機能
- 課金
- AI生成（イラスト・画像判定ともに）
- ログイン機能
- 通知
- ランキング

まずはMVP完成を優先する。
