# Future Album

目的：
親や家族との未来の思い出を作るためのアプリ。


コンセプト：
未来アルバムは画像生成アプリではない。
ユーザーが家族との未来の時間を想像し、実際の行動につなげるためのアプリである。

ターゲット：
家族、主に親の高齢化を感じ始めた人

方針：
- シンプル最優先
- 登録不要
- スマホファースト
- PWA対応を想定
- 個人情報は最小限
- MVPを優先


保存方針：
MVPではサーバー保存を行わない。
保存先はブラウザ内とする。
利用候補
IndexedDB

保存対象：
ミッション
達成状況
家族写真
AI未来写真
実際の達成写真

MVP機能：
1. ミッション登録
2. ミッション一覧表示
3. 達成チェック
4. 写真登録
5. アルバム表示

禁止：
- SNS機能
- 課金
- AI生成
- ログイン機能

まずはMVP完成を優先する。
